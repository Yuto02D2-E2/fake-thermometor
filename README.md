# fake thermometor repo

大学とかの入り口にある検温器の偽物を作ってみた

顔検出して，その近くにランダムな実数値を表示しているだけ

## Usage
```bash
$ pip install -U pip

$ pip install mediapipe
# もしも2022/08/10 @ Win10/64bitで動作確認済みのバージョンが良いなら，requirements.txtを使う
$ pip install -r requirements.txt

$ python app.py
```
